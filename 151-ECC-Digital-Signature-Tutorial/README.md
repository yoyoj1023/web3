### **詳細課程內容**

#### **模組一：數位簽章基礎 (Digital Signature Basics)**

**課程目標：** 理解數位簽章的概念、功能和應用場景。
**心智模型：** 數位簽章就像紙本文件上的親筆簽名，但在數位世界中提供更強的安全保障。

1.  **什麼是數位簽章？**
    *   **身份驗證 (Authentication)：** 確認簽署者的身份
    *   **完整性 (Integrity)：** 確保訊息沒有被修改
    *   **不可否認性 (Non-repudiation)：** 簽署者無法否認簽署行為

2.  **應用場景：**
    *   加密貨幣交易簽名（Bitcoin、Ethereum）
    *   電子合約簽署
    *   軟體發布驗證
    *   安全通訊

#### **模組二：橢圓曲線介紹 (Elliptic Curve Introduction)**

**課程目標：** 掌握橢圓曲線的基本數學概念和運算。
**心智模型：** 橢圓曲線提供了一個特殊的數學結構，讓我們能用更短的金鑰達到更高的安全性。

1.  **橢圓曲線方程式：**
    *   **標準形式：** y² ≡ x³ + ax + b (mod p)
    *   **有限體：** 所有運算都在 0 到 p-1 範圍內取模
    *   **參數條件：** 4a³ + 27b² ≠ 0 (mod p)

2.  **點加法運算：**
    *   **幾何直觀：** 兩點連線與曲線的第三個交點
    *   **代數計算：** 使用斜率公式計算新點座標
    *   **點乘法：** 重複的點加法運算

#### **模組三：ECDSA 核心概念 (ECDSA Core Concepts)**

**課程目標：** 理解 ECDSA 的基本原理和安全性基礎。
**心智模型：** ECDSA 利用橢圓曲線的數學難題（離散對數問題）來保證安全性。

1.  **公私鑰系統：**
    *   **私鑰：** 一個隨機選擇的秘密數字
    *   **公鑰：** 私鑰與基準點的乘積（一個曲線上的點）
    *   **安全性：** 從公鑰反推私鑰在計算上不可行

2.  **為什麼選擇橢圓曲線？**
    *   **效率：** 256 位元 ECC ≈ 3072 位元 RSA
    *   **性能：** 更快的運算速度
    *   **適用性：** 適合資源受限的環境

#### **模組四：簽署與驗證流程 (Signing & Verification Flow)**

**課程目標：** 深入理解 ECDSA 的完整工作流程。
**心智模型：** 簽署是用私鑰創造證明，驗證是用公鑰檢查證明的有效性。

1.  **簽署流程：**
    1.  計算訊息雜湊值 z
    2.  選擇隨機數 k
    3.  計算點 (x₁, y₁) = k * G
    4.  計算 r = x₁ mod n
    5.  計算 s = k⁻¹ * (z + r * d) mod n
    6.  輸出簽章 (r, s)

2.  **驗證流程：**
    1.  計算訊息雜湊值 z
    2.  計算 s⁻¹ mod n
    3.  計算 u₁ = z * s⁻¹ mod n
    4.  計算 u₂ = r * s⁻¹ mod n
    5.  計算點 (x₁, y₁) = u₁ * G + u₂ * Q
    6.  驗證 r ≡ x₁ mod n

#### **模組五：手算演練 (Hands-On Calculation)**

**課程目標：** 通過實際計算深化對 ECDSA 運算過程的理解。
**心智模型：** 手算過程雖然繁瑣，但能讓您真正理解每一步的數學意義。

1.  **簡化參數設定：**
    *   曲線：y² ≡ x³ + 2x + 2 (mod 17)
    *   基準點：G = (5, 1)
    *   範例私鑰：d = 6
    *   曲線階數：n = 19

2.  **完整計算範例：**
    *   演練 1：從私鑰生成公鑰
    *   演練 2：簽署訊息
    *   演練 3：驗證簽章

1.  **概念題：**
    *   數位簽章的三大功能
    *   ECC 相對 RSA 的優勢
    *   公私鑰的保密性

2.  **計算題：**
    *   點加法運算

    *   簽章驗證

3.  **應用題：**
