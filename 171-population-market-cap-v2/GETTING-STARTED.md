# 🚀 快速開始指南

歡迎來到「台灣出生人口趨勢儀表板」課程！這份指南將幫助你快速開始學習。

## 📋 課前準備

在開始課程之前，請確保你已經具備以下條件：

### 必要工具

- [ ] **Node.js**（版本 18.x 或以上）
  - 檢查方法：在終端機執行 `node -v`
  - 下載：https://nodejs.org/

- [ ] **Git**
  - 檢查方法：在終端機執行 `git --version`
  - 下載：https://git-scm.com/

- [ ] **程式碼編輯器**
  - 推薦：Visual Studio Code
  - 下載：https://code.visualstudio.com/

### 必要帳號

- [ ] **GitHub 帳號**
  - 註冊：https://github.com/signup
  - 用途：版本控制、部署整合

- [ ] **Neon 帳號**
  - 註冊：https://neon.tech
  - 用途：Serverless PostgreSQL 資料庫
  - 💡 建議使用 GitHub 帳號註冊

- [ ] **Vercel 帳號**
  - 註冊：https://vercel.com/signup
  - 用途：部署網站到正式環境
  - 💡 建議使用 GitHub 帳號註冊

### 必備知識

你應該已經了解以下內容：

- ✅ JavaScript/TypeScript 基礎語法
- ✅ React 基本概念（Components、Props、State）
- ✅ 基本的 HTML/CSS
- ✅ 終端機基本操作（cd、ls、mkdir 等）
- ✅ Git 基本概念（commit、push、pull）

**如果某些知識還不太熟悉，也沒關係！課程中會有詳細說明。**

## 🎬 開始學習

### 步驟 1：確認資料

你應該已經有 2016-2025 年的台灣出生人數資料（在圖片中）。這些資料已經整理在 `data/taiwan-births.json` 檔案中。

1. 開啟 [第一章：資料準備與前處理](./chapter-01-data-preparation.md)

如果你已經熟悉某些主題，可以直接跳到特定章節：

1. 建立專案資料夾：
   ```bash
   mkdir tw-birth-tracker
   cd tw-birth-tracker
   ```

2. 開啟你的程式碼編輯器：
   ```bash
   code .  # 如果使用 VS Code
   ```

3. 準備好你的終端機

### 步驟 4：開始第一章

前往 [第一章：資料準備與前處理](./chapter-01-data-preparation.md) 開始學習！

## 🆘 遇到問題怎麼辦？

### 步驟 1：檢查疑難排解區塊

每一章的最後都有「疑難排解」或「常見問題」區塊，先查看是否有你遇到的問題。

### 步驟 2：檢查錯誤訊息

仔細閱讀終端機或瀏覽器 Console 的錯誤訊息，通常會告訴你問題出在哪裡。

### 步驟 3：Google 搜尋

搜尋關鍵字 + 錯誤訊息，例如：
```
Next.js "Cannot find module" error
```

### 步驟 4：查看官方文件

- [Next.js Docs](https://nextjs.org/docs)
- [Neon Docs](https://neon.tech/docs)
- [Recharts Docs](https://recharts.org/)

### 步驟 5：重新開始該章節

有時候重新來過，反而比修復錯誤更快。
